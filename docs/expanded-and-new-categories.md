# TIL App Content Expansion

## Additional Topics + New Categories

These additions are designed specifically for the **Today I Learned** app philosophy:

- Topics should be interesting, memorable, and fun.
- Users receive only one topic per day.
- Each topic will contain only 3–5 facts.
- Breadth is more valuable than depth.
- Variety helps reduce repeated content and keeps daily discoveries fresh.
