[Machine learning](https://www.britannica.com/technology/machine-learning) is a process that enables [computers](https://www.britannica.com/technology/computer) to learn autonomously by identifying patterns and making data-based decisions. This approach is particularly useful in situations where it is impractical to write detailed instructions for every possible scenario. Many [expert systems](https://www.britannica.com/technology/expert-system) (programs that solve problems that normally require human expertise) and [data mining](https://www.britannica.com/technology/data-mining) programs (which discover useful patterns in large volumes of data) use machine learning to improve their [algorithms](https://www.britannica.com/science/algorithm). While machine learning is a subset of artificial intelligence, the two should not be confused as the same.. By using machine learning, a computer may attain some level of artificial intelligence.

The most popular tools used in machine learning are artificial [neural networks](https://www.britannica.com/technology/neural-network) and [genetic algorithms](https://www.britannica.com/technology/genetic-algorithm). Artificial neural networks mimic the way the human [brain](https://www.britannica.com/science/brain) operates, using weighted decision paths to process information. Genetic algorithms, on the other hand, simulate the process of [natural selection](https://www.britannica.com/science/natural-selection), in which symbols are “bred” and refined by algorithms to create increasingly effective programs. These methods allow machine learning systems to adapt and improve over time.

Related Topics: [machine learning](https://www.britannica.com/technology/machine-learning)_(Show more)_

[See all related content](https://www.britannica.com/facts/What-Is-Machine-Learning)

Machine learning is used in multiple fields, well beyond its traditional applications in expert systems and data mining. For example, many image classification systems use machine learning, in which specialized neural networks, such as convolutional neural networks (CNNs), are trained to recognize and categorize images. Additionally, machine learning is used to develop safe [autonomous vehicles](https://www.britannica.com/technology/autonomous-vehicle), improving the algorithms that enable these vehicles to navigate complex environments safely.
